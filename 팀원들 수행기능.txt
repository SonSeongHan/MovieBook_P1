정율아 (하나의 crud + spa 생성)
  전체적인 시작 틀 (두 개의 div 박스) 생성
   - 오른쪽 div 박스 (aka. sns 박스) 생성 및 전체 제작
   - sns 박스
  기존 포스터 클릭 시 해당 포스터로 이동
   - 좋아요, 좋아요 취소 가능
   - 해당 포스터 삭제 가능 (비밀번호 입력 alert 뜸)
   - 게시글 내용 수정, 댓글달기 기능 (crud)
   - 댓글 내용 수정 및 삭제 가능
   - back to post 클릭 시 전화면으로 이동
   - 제일 위 New Post 클릭 시 새로운 포스트가 제일 위쪽에 저장
  기존에 있는 sns 글 기능 api로 받아오기
  TMDB API (review 한정) 받아옴

손성한(영화 박스 전반적인 틀 구성 및 기본적 기능 구현)
  왼쪽 div박스 틀 제작(aka. 영화 박스)
  영화 박스
   - 첫배경화면 깔아두기(z-index:-1)
   - 추천, 대여, 구매 박스 생성
   - 해당 박스 클릭 시 화면 전환
  추천
   - 최신, 추천, 장르를 담을 수 있는 div박스 생성
   - 영화 목록 가로로 배열
   - 장르를 선택할 수 있는 박스 틀 생성
   - 특정 영화 클릭 시 새로운 페이지에 해당 영화 상세정보창 띄움
  이동
   -해당영화 클릭 시 상세 페이지로 이동
   -상세 페이지에서 구매, 대여 버튼 생성, 리뷰 기능 추가(crud) ex) 추가 수정 삭제 가능
   -이동된 해당 영화들 각각 페이지로 이동 가능하게 하고 그 안에서 환불로 삭제 가능

이진수(API 담당)
  필요한 영화의 정보들을 가져올 API 따옴
   - TMBD API 사용
  영화를 필요에 따른 정보들로 구성하여 받아옴
   - 최신 영화 : 개봉순
   - 추천 영화 : 평점순
   - 장르 선택 : 장르 선택 박스에 장르를 선택하면 해당 장르에 맞는 작품들을 띄워줌
   - 영화 목록 리스트들을 보기 좋게 정렬(css)
  검색
   - 각 페이지 별로 검색 기능 추가
   - 이름이 하나라도 겹치면 그에 관련된 모든 영화 리스트 보여줌
   - 대여중인 영화와 구매한 영화 페이지에는 그 안에 있는 영화들 중에서만 뜨게 기능
  이동
   - 주소값이 같은 상태에서 뒤로가기를 누르면 홈페이지가 아닌 바로 전에 봤던 페이지로 이동
